# Dungeon Archivist ML Project for Revature AI Engineering Course
#### Group 2: Jose, Lawson, and Stephen
Uses Keras, TensorFlow, and TensorBoard to analyze video game sprite images and automatically categorize them based on the training data.

Training Data from [Dungeon Crawl Stone Soup](https://opengameart.org/sites/default/files/Dungeon%20Crawl%20Stone%20Soup%20Full_0.zip).
